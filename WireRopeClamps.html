Product Overview

The Crosby® G-450 Red U-Bolt® Clip is engineered for superior performance in demanding wire rope applications. These hot-dip galvanized wire rope clips provide maximum strength and corrosion resistance for critical connections.

Efficiency: 80% for 3mm-22mm sizes | 90% for 24mm-90mm sizes

Product Identification Code (PIC) on each base

Certified

Durable

Hot-dip galvanized for corrosion resistance

Forged

Forged bases for sizes 3mm-62mm and 75mm

Key Features

  • Each base has a Product Identification Code (PIC) for material traceability
  • Name Crosby or "CG" and size forged into each base
  • 80% efficiency rating for 3mm-22mm sizes, 90% for 24mm-90mm sizes
  • Entire clip is galvanized to resist corrosion and rust
  • Sizes 3mm-62mm and 75mm have forged bases
  • All clips individually bagged or tagged with application instructions and warnings
  • Clip sizes up to 38mm have rolled threads
  • Meets or exceeds all requirements of ASME B30.26
  • Look for the Red U-Bolt® for genuine Crosby Clips

Technical Specifications
